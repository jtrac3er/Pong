Pong v1.2

####################### Installation ####################

Um zu funktionnieren, muss Python installiert sein (Version >=3.6). Programmiert wurde mit Version 3.6.1., es
kann daher Kompatibilitätsprobleme geben.

Das Spiel hat folgende Abhängigkeiten:
 - Pygame
 - NumPy

Diese müssen installiert sein. Sie können unter Windows auf die Datei "requirements.bat" drücken, um die 
Abhänigkeiten zu installieren. Beachten Sie, dass pip im Pfad liegen muss!

###################### Spiel ##########################

Gespielt wird mit den Pfeiltasten. Viele Einstellungsmöglichkeiten stehen Ihnen im Menu (Zahnrad) zur Verfügung.
Um im Spiel zu pausieren, drücken Sie die linke Maustaste. 

Viel Spass bei Spielen
